概览

vxServer 可以将您的 Github 仓库变为静态服务器。简单的说，具有如下几个优点：

  • 灵活。您可以自由设置将哪个分支作为静态服务器的根目录，可以绑定自己的域名，还可以设置 SSL 的支持(即将实装此功能)。
  • 借助微林 CDN 网络，相比 Github Pages 或者其它服务，访问速度更快。
  • 完全实时。vxServer 与 Github 仓库实时同步，这无需单独设置 webhook 。
  • 免费的 *.code.so 二级域名。
  • 完全免费。

创建项目

请先 登录 微林

请先 回到后台界面

点击左侧导航的"静态托管"按钮进入服务页面。

请点 “创建新的托管项目”。

在您要托管的 Github 仓库的目标分支下，创建 “verification.txt”，其内容为 “Github 验证文件” 一栏中提供的内容。

设定 “名称”、“域名”、“Github 项目地址”。点击 “创建”。

创建完成后，请移除 “verification.txt”。

我没有自己的域名怎么办

vxServer 为大家提供了 code.so 的二级域名。
在创建项目时，可以自己填写一个喜欢的域名，比如 “caddy.code.so”。
