先知实时监测系统

分布式，多维度。即将到来。

先知实时检测系统是一项分布式服务器性能检测系统，由主动探针与微林 API 服务组成。

    探针
  • 开放源代码
  • 自行部署
  • 支持 Linux 系统
  • 秒级检测数据，系统资源占用低
  • 监测指标：CPU 使用率，系统负载，网络，磁盘使用率
    微林 API 服务
  • 收集并整理探针数据
  • 提供最近1000次实时数据
  • 自定义的监测面板，实时查阅监测数据
  • 提供最近3个月归档历史数据
  • Telegram，Webhook，邮件三种告警方式
